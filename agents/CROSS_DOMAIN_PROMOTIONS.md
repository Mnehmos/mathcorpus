# Cross-Domain Promotions

Proposals to promote a lemma, tactic pattern, or kit from a single domain's
`CROSS_DOMAIN.md` into shared infrastructure (a new/extended kit under
`lean/MathCorpus/`, declared via `dependencies.kits`).

| Date | Proposed by (domain) | Pattern | Target kit | Status |
|------|--------------------------|---------|-------------|--------|
|      |                          |         |             |        |

A proposal is resolved when the kit exists in `lean/MathCorpus/` and the
originating packets are updated to depend on it via `dependencies.kits`.
